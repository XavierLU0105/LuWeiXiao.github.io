橙子 33 2.2 富含类黄酮、果胶等，有助于增强免疫力、促进消化，对心血管健康有益。其所含的类黄酮具有抗氧化作用，能减轻身体的炎症反应；果胶可调节肠道微生物群落，促进肠道健康；丰富的钾元素有助于维持正常的血压水平和心脏功能。此外，橙子还含有香豆素类化合物，具有一定的抗癌活性；橙皮苷能改善血管通透性，降低血脂；橙子中的有机酸可促进食欲，帮助消化吸收其他食物中的营养成分。
苹果 4 2.4 含有丰富的果胶和抗氧化物质，能帮助降低胆固醇，维持肠道健康，具有一定的抗氧化作用。苹果中的果胶可降低低密度脂蛋白胆固醇水平，减少心血管疾病风险；抗氧化物质如槲皮素等有助于保护细胞免受自由基损伤，预防慢性疾病；还含有硼元素，对骨骼健康有益。苹果还富含苹果酸等有机酸，能调节身体酸碱平衡，促进新陈代谢；其含有的多酚类物质具有抗炎和抗菌特性，有助于预防口腔疾病和肠道感染；同时，苹果的香气成分有助于舒缓情绪，减轻压力。
香蕉 8 2.6 富含钾元素，有助于维持心脏正常功能和血压稳定，还含有一定量的维生素 B6，对神经系统有益。每 100 克香蕉含钾量高达 300 - 400 毫克，能调节体内电解质平衡；维生素 B6 参与多种神经递质的合成，有助于维持神经系统正常功能；同时含有丰富的碳水化合物，能快速提供能量，缓解疲劳。香蕉还含有丰富的镁元素，能放松肌肉，缓解焦虑；其含有的色氨酸可转化为血清素，有助于改善睡眠质量；此外，香蕉中的抗性淀粉在肠道内可被有益菌发酵，产生短链脂肪酸，维护肠道健康。
草莓 47 2 含有大量的花青素等抗氧化物质，具有抗氧化、抗炎作用，对皮肤健康有益，还富含叶酸等营养成分。花青素能有效清除自由基，预防衰老和多种慢性疾病；草莓中的叶酸对于孕妇胎儿的神经管发育至关重要；此外，草莓还含有多种有机酸，有助于促进消化液分泌，增进食欲。草莓还含有鞣花酸，具有抗癌、抗菌和抗病毒的潜在功效；其丰富的维生素 B 族成分参与身体的能量代谢和神经系统功能维持；草莓中的果胶和纤维素有助于降低胆固醇吸收，促进肠道蠕动，预防便秘和结肠癌。
葡萄 25 0.9 含有的白藜芦醇等成分具有抗氧化、预防心血管疾病的潜在作用，同时富含葡萄糖、矿物质等，能快速补充能量。白藜芦醇可降低血液黏稠度，抑制血小板聚集，预防血栓形成；葡萄中的葡萄糖能迅速被人体吸收利用，提供即时能量；此外，还含有多种维生素和矿物质，如维生素 K、钙、铁等，有助于维持身体正常生理功能。葡萄中的原花青素含量较高，具有比维生素 C 和 E 更强的抗氧化能力，能保护眼睛晶状体和视网膜，预防眼部疾病；葡萄皮和籽中含有的黄酮类化合物对心血管系统具有保护作用，可降低冠心病的发病风险；此外，葡萄中的酒石酸等有机酸有助于促进消化，增进食欲。
西瓜 6 0.3 水分含量高，能补充水分和电解质，还含有一定量的番茄红素，具有抗氧化和保护心血管的功效。西瓜的水分含量可达 90%以上，在炎热天气中能有效补充因出汗而流失的水分和电解质；番茄红素具有很强的抗氧化能力，可降低前列腺癌等疾病的风险；同时含有少量的维生素 B1、维生素 B6 和镁等营养成分，有助于维持身体正常代谢。西瓜还含有瓜氨酸，可促进血管舒张，降低血压；其富含的维生素 C 除了抗氧化作用外，还参与胶原蛋白的合成，对皮肤健康有益；此外，西瓜中的钾元素有助于维持心脏正常节律和肌肉功能。
蓝莓 9.7 2.4 富含花青素、维生素 K 和锰等营养成分。花青素具有卓越的抗氧化性能，能保护眼睛视网膜细胞，预防眼部疾病如黄斑病变和白内障；维生素 K 对骨骼健康至关重要，参与钙的代谢和骨骼形成；锰元素有助于维持正常的血糖水平和神经系统功能，还在抗氧化防御系统中发挥作用。蓝莓中的植物化学物质如绿原酸等具有抗炎和抗菌特性，有助于预防尿路感染和慢性炎症；其富含的膳食纤维有助于调节肠道菌群，改善肠道健康；此外，蓝莓中的维生素 E 和锌等营养成分对维持身体正常的免疫功能和生殖系统健康具有重要作用。
猕猴桃 62 2.6 除了高含量的维生素 C 外，还含有维生素 E、钾、镁和叶酸等多种营养物质。维生素 C 和 E 协同作用，具有强大的抗氧化能力，能增强免疫力、预防心血管疾病和癌症；丰富的钾元素有助于维持正常血压；镁元素参与身体多种酶的反应，对心脏和神经系统健康有益；叶酸对于孕妇和胎儿的健康发育非常重要。猕猴桃中的精氨酸可促进血液循环，改善心血管功能；其含有的叶黄素和玉米黄质对眼睛具有保护作用，可预防视网膜病变；此外，猕猴桃中的蛋白酶有助于消化蛋白质，减轻胃肠负担，对消化不良和腹胀等问题有一定缓解作用。
芒果 23 1.6 富含维生素 A（以β-胡萝卜素形式存在）、维生素 C 和膳食纤维。维生素 A 对视力保护至关重要，有助于维持正常的视觉功能和上皮组织健康；维生素 C 具有抗氧化和免疫增强作用；膳食纤维能促进肠道蠕动，预防便秘；芒果还含有芒果苷等生物活性成分，具有抗炎、抗菌和抗氧化等潜在功效。芒果中的芒果酮酸等成分具有抗癌活性；其丰富的维生素 B6 有助于维持神经系统正常功能和血红蛋白合成；芒果中的铁元素含量虽不高，但在与维生素 C 协同作用下，能提高铁的吸收率，对预防缺铁性贫血有一定帮助；此外，芒果的香气成分具有独特的舒缓身心的作用，可缓解焦虑和压力。
梨 6 3.1 含有丰富的果胶、硼元素和多种维生素。果胶有助于降低胆固醇水平，促进肠道健康；硼元素对骨骼健康有益，可增强骨密度；梨中的维生素 B2、维生素 B3 和维生素 C 等参与身体的新陈代谢和抗氧化防御系统；此外，梨的水分含量高，具有润肺止咳、生津止渴的功效。梨还含有木质素等成分，具有一定的防癌抗癌作用；其含有的果糖和葡萄糖易于消化吸收，可为身体快速提供能量；梨中的有机酸能促进胃肠蠕动，帮助消化，缓解便秘；同时，梨的寒性特质使其在清热降火方面具有一定作用，适合在燥热天气食用。
柠檬 22 1.3 富含维生素 C、柠檬酸、类黄酮等营养成分。维生素 C 具有强大的抗氧化作用，能增强免疫力，促进胶原蛋白合成；柠檬酸可促进胃液分泌，帮助消化，还能与钙离子结合，预防肾结石形成；类黄酮具有抗炎、抗菌和抗病毒功效，有助于预防心血管疾病和某些癌症。柠檬中的柠檬烯具有独特的香气，能舒缓情绪，减轻焦虑和压力；其含有的钾元素有助于维持心脏正常功能和血压稳定；此外，柠檬还可作为天然的防腐剂，用于食品保鲜和调味，增添食物的风味。
菠萝 18 1.3 含有丰富的维生素 C、锰、菠萝蛋白酶等营养成分。维生素 C 具有抗氧化和免疫增强作用；锰元素参与身体多种酶的反应，对骨骼发育和新陈代谢有益；菠萝蛋白酶有助于分解蛋白质，促进消化吸收，减轻胃肠负担，还具有抗炎和消肿的作用，可缓解关节炎等炎症性疾病症状。菠萝中的膳食纤维能促进肠道蠕动，预防便秘；其含有的维生素 B1 对维持神经系统正常功能和碳水化合物代谢具有重要作用；此外，菠萝的独特风味能刺激食欲，增加唾液和胃液分泌，有助于开胃和消化。
火龙果 9 1.6 富含维生素 C、膳食纤维、铁等营养成分。维生素 C 具有抗氧化作用；膳食纤维有助于促进肠道蠕动，预防便秘，降低胆固醇吸收；火龙果中的铁元素含量较高，对于预防缺铁性贫血具有一定作用，尤其是红心火龙果中的天然色素具有抗氧化和抗炎特性，可保护细胞免受自由基损伤；火龙果还含有植物白蛋白，能与体内的重金属离子结合，将其排出体外，起到解毒作用；此外，火龙果中的芝麻状种子富含不饱和脂肪酸，对心血管健康有益，还能增加饱腹感，有助于控制体重。
柚子 23 1.2 富含维生素 C、类黄酮、柚皮苷等营养成分。维生素 C 具有抗氧化和免疫增强作用；类黄酮和柚皮苷具有抗炎、抗菌和降低胆固醇的功效，有助于预防心血管疾病和癌症；柚子中的果胶能促进肠道健康，调节血糖和血脂水平；其含有的钾元素有助于维持心脏正常功能和血压稳定；此外，柚子的清香气味能缓解焦虑和压力，促进睡眠；柚子还具有一定的减肥功效，其低热量、高纤维的特点能增加饱腹感，减少其他高热量食物的摄入。
